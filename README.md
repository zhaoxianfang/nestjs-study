# nestjs 学习项目

## Installation
```
$ npm install
```
## Running the app
```
$ npm run start:dev
```
# development
```
$ npm run start
```
# watch mode
```
$ npm run start:dev
```
# production mode
```
$ npm run start:prod
```

## Test

# unit tests
```
$ npm run test
```
# e2e tests
```
$ npm run test:e2e
```
# test coverage

```
$ npm run test:cov
```
